# DrinkHappie Website Assets

Backup of all custom-built and modified Shopify theme files for [drinkhappie.com](https://drinkhappie.com).

## Sections

| File | Description |
|------|-------------|
| `amazon-reviews.liquid` | Amazon reviews carousel with cream-colored cards |
| `blog-posts.liquid` | Homepage blog carousel (Splide auto-scroll right) |
| `main-blog.liquid` | Blog listing page — Pinterest-style masonry layout with hover glow |
| `blog-posts-wide.liquid` | Wide horizontal blog card layout (unused, kept for future) |
| `article-shop-cta.liquid` | "Shop the Lineup" product grid on blog articles |
| `variety-pack-promo.liquid` | "Find Your Flavor" variety pack section with hover pop-up |
| `choose-your-vibe.liquid` | THC Seltzers vs Fungi Fusion comparison cards with hover pop-up |

## Snippets

| File | Description |
|------|-------------|
| `happie-product-card.liquid` | Standardized product card component |
| `happie-product-card-styles.liquid` | Product card CSS styles |

## Templates

| File | Description |
|------|-------------|
| `index.json` | Homepage template config |
| `blog.json` | Blog listing page template config |
| `article.json` | Blog article page template config |

## Theme

Built on **Shapes** by Switch Themes, deployed via GitHub to Shopify.
